*** 1.1 使用注解

注解是一种可以插到源码中的标签，这些标签可以使用一些工具对其进行相应的处理

+ 注解是一种修饰符
+ 用来配合反射
  + 通过注解来获取想要的方法、成员或者参数，进而对其进行操作
