* 关于NDN中控制访问的总结

+ 范围
  + 集中于一定的场景中
  + 控制住范围,不要妄想得太多
+ 目标
  + 先关注一部分控制访问的基础实现策略
+ 注意
  + 实现方式多种多样,不要被当前的解决方案限制住思维

** 综述

+ 目的
  + 防止未被授权的用户访问受保护的网络资源
  + 允许被授权的用户合法第访问网络资源
+ NDN下访问控制思路
  + 基于命名的访问控制
  + 基于兴趣包的访问控制
  + 基于加密的访问控制
+ 问题
  + 这得要做这方面么
  + 如果基于一个大型的直播场景
    + 兴趣包的聚合效率有多少?
      + 设计一个场景
      + 开始实验
      + 得出结果
