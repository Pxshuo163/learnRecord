** 1.基本概念
*** 1.1 基本概念
**** 知识体系
**** R简介
***** S语言的一种实现,进行数据探索,统计分析
***** 拥有非常丰富的函数工具
*** 1.2 相关命令
**** 创建向量与矩阵
***** 数组 :: c()
***** 变量类型 :: mode(x)
**** 矩阵
***** rbind(x1, x2) :: 合并成两行
***** cbing(x1, x2) :: 合并成两列
***** 标准差 :: sd
***** 方差 :: var(x)
***** 连乘 :: prod()
***** 帮助文档 :: help()
***** 赋值 :: <-
***** 1-100*2+1 :: 1到100,每个数字都做*2+1的操作
***** 索引 :: a[3]
***** seq(2,6,by=2) :: 2,4,6
***** seq(2,6,length=5) :: 2,3,4,5,6
